## 1. 单位属性规范

所有单位共享以下属性分类。塔、士兵、机关、敌人、BOSS本质相同，只是配置值不同。

| 分类 | 字段 | 类型 | 说明 | 谁有 |
|------|------|------|------|------|
| **标识** | `id` | string | 配置主键（snake_case） | 所有 |
| | `name` | string | 显示名 | 所有 |
| | `category` | enum | Tower/Soldier/Trap/Enemy/Boss | 所有 |
| **生存** | `hp` | number | 最大生命值 | 所有 |
| | `armor` | number | 护甲值 | 所有 |
| | `magicResist` | number | 魔抗值 | 所有 |
| **战斗** | `atk` | number | 攻击力 | 有攻击能力的 |
| | `attackSpeed` | number | 每秒攻击次数 | 有攻击能力的 |
| | `range` | number | 攻击射程（px） | 有攻击能力的 |
| | `damageType` | enum | physical/magic/true | 有攻击能力的 |
| | `projectileType` | enum | instant/arrow/beam/missile/aoe | 有攻击能力的 |
| **机动** | `moveSpeed` | number | 移动速度（px/s），0=不可移动 | 可移动单位 |
| | `moveRange` | number | 移动范围半径（px） | 士兵 |
| **经济** | `cost` | number | 建造/部署消耗（仅参考，由卡牌控制） | 玩家单位 |
| | `killReward` | number | 击杀奖励金币 | 敌人 |
| | `upgradeCosts` | number[] | 各等级升级费用 | 塔 |
| **层级** | `layer` | enum | Grid/Ground/LowAir/Space | 所有 |
| **阵营** | `faction` | enum | Justice/Evil/Chaos/Neutral | 所有 |
| **视觉** | `color` | string | 主色调（hex） | 所有 |
| | `shape` | string | 几何体描述 | 所有 |
| | `size` | number | 渲染尺寸（px） | 所有 |
| **机制** | `attackMode` | enum | nearest/closest_to_crystal/lowest_hp/highest_threat | 有攻击能力的 |
| | `specialEffects` | string[] | 特殊机制标签 | 部分单位 |
| | `skill` | string | 技能引用 | 部分单位 |

## 2. 塔类单位

塔固定在地面层，不可移动，自动攻击射程内敌人。每座塔有升级路径，升级改变外观+提升属性+可能解锁新机制。

### 2.1 箭塔

> 基础单体物理输出，攻速快、单发伤害低。

| 等级 | 名称 | 攻击力 | 攻速 | 射程 | 特殊机制 | 升级费 |
|------|------|--------|------|------|---------|--------|
| L1 | 普通箭塔 | 15 | 2.5/s | 180px | 单发 | - |
| L2 | 双重箭塔 | 15(×2) | 2.5/s | 180px | 同时射2箭，可锁不同目标 | 80 |
| L3 | 三重箭塔 | 15(×3) | 2.5/s | 200px | 同时射3箭，可锁不同目标 | 150 |

- 伤害类型: physical
- 弹道类型: arrow（直线飞行箭矢）
- 目标选择: closest_to_crystal
- 视觉: 棕色三角形塔体+弓弦装饰，升级增加箭筒数量

### 2.2 弩塔

> 穿透型直线AOE物理输出，发射蓝色脉冲弩箭贯穿路径上所有敌人，飞至棋盘边缘消失。

| 等级 | 名称 | 攻击力 | 攻速 | 射程 | 特殊机制 | 升级费 |
|------|------|--------|------|------|---------|--------|
| L1 | 普通弩塔 | 45 | 0.8/s | 220px | 穿透：弩箭沿直线飞行，贯穿路径上所有敌人，飞至棋盘边缘消失 | - |
| L2 | 高级弩塔 | 65 | 0.8/s | 220px | 穿透+伤害提升 | 100 |
| L3 | 超级弩塔 | 65 | 1.2/s | 240px | 穿透+伤害提升+攻速提升 | 180 |

- 伤害类型: physical
- 弹道类型: arrow（直线贯穿——弩箭沿锁定方向飞行，不做目标追踪）
- 目标选择: closest_to_crystal（优先射向敌人最密集的直线方向）
- 视觉: 蓝色脉冲弩箭——箭头为纯蓝色(#2196f3)，箭身从白渐变为蓝，整体尺寸为普通箭矢3倍，飞行速度较慢。塔体为蓝色脉冲弩机造型，升级增大弩臂

## 4. 机关单位

机关是被动型单位，通常放置在路径上或路径旁。

**视觉设计原则** ：所有机关使用几何图形拼接绘制，风格统一且辨识度高。

### 4.1 地刺

| 属性 | 值 |
|------|-----|
| 伤害 | 8/s |
| 层级 | Grid（地格层） |
| 放置限制 | 必须在路径格上 |
| 效果 | 对经过的地面层敌人每秒造成伤害 |
| **视觉** | **3个灰色三角形并排排列，触发动画时向上弹出** |

### 4.2 捕兽夹

| 属性 | 值 |
|------|-----|
| 效果 | 触发后困住敌人1秒 |
| 层级 | Grid |
| 放置限制 | 必须在路径格上 |
| 限制 | 对BOSS无效 |
| 触发方式 | 一次性，触发后消失 |
| **视觉** | **两个棕色菱形左右组合表示金属夹子，中心矩形锯齿装饰** |

### 4.3 焦油坑

| 属性 | 值 |
|------|-----|
| 减速效果 | 20% |
| 层级 | Grid |
| 放置限制 | 必须在路径格上 |
| 效果 | 经过的地面层敌人持续减速 |
| **视觉** | **黑色大圆形（半透明）+ 深色小圆形内圈，表现焦油粘稠感** |

### 4.4 巨石

| 属性 | 值 |
|------|-----|
| HP | 200 |
| 护甲 | 20 |
| 层级 | Ground |
| 放置限制 | 必须在路径格上 |
| 效果 | 阻挡地面敌人前进，敌人必须攻击破坏后才能继续 |
| 可否被攻击 | ✅（敌人会攻击巨石） |
| **视觉** | **灰色大圆形主体 + 两道深色矩形裂纹装饰** |

> **注** ：机关种类现调整为4种（地刺/捕兽夹/焦油坑/巨石）。风扇、水坑、拳击套、机械臂已移除。

## 6. BOSS

> BOSS统一标记 `category: Boss`，拥有专用AI和特殊技能。BOSS不受捕兽夹影响。

### 6.0 敌人技能完整性规则

敌人配置必须满足以下技能完整性要求：

- 精英怪：所有 `tier: L2` 的敌人视为精英怪，必须配置且仅配置 1 个技能。
- Boss：所有 `isBoss: true` 或 `category: Boss` 的敌人视为 Boss，必须配置 1 个以上技能。
- 技能字段统一使用 `skills` 数组，每项至少包含 `id`、`name`、`cooldown`、`range`、`value`、`description`。
- 当前运行实现中，`EnemySkillSystem` 会自动执行挂载 Boss 组件的 YAML 技能；精英怪技能属于设计与配置要求，若需要战斗中主动释放，后续必须补齐精英技能执行系统或接入生命周期规则。

当前敌人配置审查结论：

- Boss 缺口：`giant_slime`、`queen_beetle`、`lucifer`、`super_robot`、`abyss_lord` 尚未配置 `skills`。
- 精英怪缺口：`heavy`、`mage`、`exploder`、`e_burrow_worm`、`e_giant_beetle`、`e_frost_marauder`、`e_yeti_charger`、`e_blizzard_sprite`、`e_brine_charger`、`e_steam_lancer`、`e_hammergrub_drone`、`e_minecart_juggernaut`、`e_myco_charger`、`e_brood_mother`、`e_blight_spitter`、`e_frost_lancer`、`e_blood_trickster` 尚未配置精英技能。

## 8. 数值设计原则

> **注意** ：上述所有数值为设计初值。实际实现时需通过可玩性测试调优。

### 8.1 数值关系参考

| 关系 | 公式/说明 |
|------|----------|
| 塔DPS ≈ 敌人HP | 同等级塔DPS应能处理同等级敌人 |
| 升级价值 | 每级升级费约等于2-3波的金币收入，属性提升30-60% |
| BOSS挑战 | BOSS HP ≈ 同关普通敌人HP×10-15 |
| 精英价值 | 精英HP×2 ATK×1.5，击杀奖励吸引力匹配卡牌价值 |
| 金币节奏 | 每波金币收入应支持升级1-2座塔一次 |
