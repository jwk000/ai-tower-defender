## 1. 单位属性规范

所有单位共享以下属性分类。塔、士兵、机关、敌人、BOSS本质相同，只是配置值不同。

| 分类 | 字段 | 类型 | 说明 | 谁有 |
|------|------|------|------|------|
| **标识** | `id` | string | 配置主键（snake_case） | 所有 |
| | `name` | string | 显示名 | 所有 |
| | `category` | enum | Tower/Soldier/Trap/Enemy/Boss | 所有 |
| **生存** | `hp` | number | 最大生命值 | 所有 |
| | `armor` | number | 护甲值 | 所有 |
| | `magicResist` | number | 魔抗值 | 所有 |
| **战斗** | `atk` | number | 攻击力 | 有攻击能力的 |
| | `attackSpeed` | number | 每秒攻击次数 | 有攻击能力的 |
| | `range` | number | 攻击射程（px） | 有攻击能力的 |
| | `damageType` | enum | physical/magic/true | 有攻击能力的 |
| | `projectileType` | enum | instant/arrow/beam/missile/aoe | 有攻击能力的 |
| **机动** | `moveSpeed` | number | 移动速度（px/s），0=不可移动 | 可移动单位 |
| | `moveRange` | number | 移动范围半径（px） | 士兵 |
| **经济** | `cost` | number | 建造/部署消耗（仅参考，由卡牌控制） | 玩家单位 |
| | `killReward` | number | 击杀奖励金币 | 敌人 |
| | `upgradeCosts` | number[] | 各等级升级费用 | 塔 |
| **层级** | `layer` | enum | Grid/Ground/LowAir/Space | 所有 |
| **阵营** | `faction` | enum | Justice/Evil/Chaos/Neutral | 所有 |
| **视觉** | `color` | string | 主色调（hex） | 所有 |
| | `shape` | string | 几何体描述 | 所有 |
| | `size` | number | 渲染尺寸（px） | 所有 |
| **机制** | `attackMode` | enum | nearest/closest_to_crystal/lowest_hp/highest_threat | 有攻击能力的 |
| | `specialEffects` | string[] | 特殊机制标签 | 部分单位 |
| | `skill` | string | 技能引用 | 部分单位 |

## 2. 塔类单位

塔固定在地面层，不可移动，自动攻击射程内敌人。每座塔有升级路径，升级改变外观+提升属性+可能解锁新机制。

### 2.1 箭塔

> 基础单体物理输出，攻速快、单发伤害低。

| 等级 | 名称 | 攻击力 | 攻速 | 射程 | 特殊机制 | 升级费 |
|------|------|--------|------|------|---------|--------|
| L1 | 普通箭塔 | 15 | 2.5/s | 180px | 单发 | - |
| L2 | 双重箭塔 | 15(×2) | 2.5/s | 180px | 同时射2箭，可锁不同目标 | 80 |
| L3 | 三重箭塔 | 15(×3) | 2.5/s | 200px | 同时射3箭，可锁不同目标 | 150 |

- 伤害类型: physical
- 弹道类型: arrow（直线飞行箭矢）
- 目标选择: closest_to_crystal
- 视觉: 棕色三角形塔体+弓弦装饰，升级增加箭筒数量

### 2.2 弩塔

> 穿透型单体物理输出，攻速慢、单发伤害高。

| 等级 | 名称 | 攻击力 | 攻速 | 射程 | 特殊机制 | 升级费 |
|------|------|--------|------|------|---------|--------|
| L1 | 普通弩塔 | 45 | 0.8/s | 220px | 穿透：箭矢直线飞行，伤害路径上所有敌人 | - |
| L2 | 高级弩塔 | 65 | 0.8/s | 220px | 穿透+伤害提升 | 100 |
| L3 | 超级弩塔 | 65 | 1.2/s | 240px | 穿透+伤害提升+攻速提升 | 180 |

- 伤害类型: physical
- 弹道类型: arrow（直线穿透）
- 目标选择: closest_to_crystal（优先射向敌人最密集的直线方向）
- 视觉: 灰色重型弩机造型，升级增大弩臂

> AOE物理范围伤害，溅射攻击。

|------|------|--------|------|------|---------|---------|--------|

| L3 | 超级炮塔 | 50 | 1.0/s | 180px | 80px | AOE + 眩晕 + 击退2格 | 200 |

- 伤害类型: physical
- 弹道类型: aoe（抛物线炮弹落地爆炸）
- 目标选择: closest_to_crystal
- 视觉: 黑色圆形炮管，升级炮管加粗+增加装甲板

## 8. 数值设计原则

> **注意**：上述所有数值为设计初值。实际实现时需通过可玩性测试调优。

### 8.1 数值关系参考

| 关系 | 公式/说明 |
|------|----------|
| 塔DPS ≈ 敌人HP | 同等级塔DPS应能处理同等级敌人 |
| 升级价值 | 每级升级费约等于2-3波的金币收入，属性提升30-60% |
| BOSS挑战 | BOSS HP ≈ 同关普通敌人HP×10-15 |
| 精英价值 | 精英HP×2 ATK×1.5，击杀奖励吸引力匹配卡牌价值 |
| 金币节奏 | 每波金币收入应支持升级1-2座塔一次 |
