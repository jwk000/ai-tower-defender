# 01 — 游戏概念

> **定位**: 产品层 — 读完就能想象游戏长什么样
> **依赖**: 无（入口文档）
> **被引用**: 02-gameplay, 03-units, 04-levels

---

## 1. 游戏定位

**卡牌驱动的关卡制塔防游戏。** 在传统塔防基础上融入卡牌手牌机制和轻Roguelike元素。

核心体验三要素：
- **策略布阵** — 拖卡部署塔/士兵/机关到战场格子上，构建防线
- **卡牌抉择** — 手牌4槽位有限，杀精英3选1抽卡，每次选择改变战术
- **关卡推进** — 5个主题关卡，每关新敌人新BOSS，通关解锁新内容

---

## 2. 核心循环

```
[主菜单] → 选择关卡
    │
    ▼
[开局]：从已解锁卡池随机抽取初始手牌（4张）
    │
    ▼
┌─────────────────────────────────────────────┐
│              单关核心循环                      │
│                                               │
│  [波次开始] → 敌人从出生点沿路径向水晶移动      │
│       │                                       │
│       ├→ 玩家拖卡部署单位/释放技能              │
│       │   - 单位卡 → 放到场景格子变成实体       │
│       │   - 技能卡 → 拖到目标区域立即释放       │
│       │   - 奥术卡 → 选择后立即生效            │
│       │                                       │
│       ├→ 单位自动战斗（攻击/移动/AI）           │
│       │                                       │
│       ├→ 击杀敌人获得金币                      │
│       │                                       │
│       ├→ 每波有1个精英怪                       │
│       │   └→ 击杀 → 3选1抽卡（补入/替换手牌）  │
│       │                                       │
│       ├→ 全部敌人死亡 → 波次通过               │
│       │                                       │
│       └→ 最后一波出现BOSS                      │
│                                               │
│  [关卡通过] → 获得金币 → 进入关间强化选择      │
└─────────────────────────────────────────────┘
    │
    ▼
[关间]：从2个随机Buff中选择1个带入下一关
    │
    ▼
[通关解锁]：新卡牌加入永久卡池 + 解锁下一关
```

### 关键节奏

- **波间无缝** ：一波敌人全灭后短暂间隙（2-3秒），下一波自动开始
- **部署时机** ：任何时刻都可以拖卡部署，包括战斗中
- **塔升级** ：任何时刻可以点击塔花费金币升级

---

## 3. 资源系统

| 资源 | 范围 | 来源 | 用途 |
|------|------|------|------|
| **金币** | 单关内 | 击杀敌人、波次奖励 | 升级塔、出售塔回收50% |
| **手牌槽位** | 单关内 | 初始4张 | 部署单位/释放技能的上限 |
| **水晶HP** | 单关内 | 关卡初始值 | 归零=关卡失败，需重打 |

> 设计意图：单一货币（金币）降低认知负担。金币只在本关使用，不通关间继承。

---

## 4. 胜负条件

### 胜利
当前关所有波次敌人全灭（含BOSS）→ 关卡通过

### 失败
水晶HP归零 → 关卡失败 → 返回关卡选择界面，可重新挑战

> 水晶位于地图左侧，敌人沿路径走到水晶位置后对水晶造成伤害。详见 [02-gameplay §1.3](./02-gameplay.md#13-水晶)

## 5. 游戏流程状态机

```
MainMenu
  │
  └→ LevelSelect（5关，初始仅第1关解锁）
       │
       └→ PreBattle（开局抽初始手牌4张）
            │
            └→ Battle（关内循环）
                 │  ├→ WaveActive（敌人移动+战斗）
                 │  ├→ EliteKilled（弹出3选1抽卡面板）
                 │  ├→ WaveCleared（波次通过，短暂间隙）
                 │  └→ BossFight（最后一波）
                 │
                 ├→ LevelVictory（关卡通过）
                 │    └→ InterLevelChoice（2选1 Buff）
                 │         └→ 返回LevelSelect（下一关解锁）
                 │
                 └→ LevelDefeat（水晶归零）
                      └→ 返回LevelSelect（可重试）
```

### 各状态说明

| 状态 | 玩家能做什么 |
|------|------------|
| `MainMenu` | 开始游戏、选择关卡 |
| `LevelSelect` | 鼠标划过已解锁关卡自动选中并切换主题背景，点击关卡进入 |
| `PreBattle` | 查看初始手牌（自动抽取，不可操作） |
| `WaveActive` | 拖卡部署、点击塔升级/出售 |
| `EliteKilled` | 3选1抽卡面板弹出，选择一张 |
| `WaveCleared` | 查看金币收益，等待下一波 |
| `BossFight` | 同WaveActive，但是最后一波 |
| `LevelVictory` | 查看结算（击杀数/金币等） |
| `InterLevelChoice` | 2选1 Buff选择面板 |
| `LevelDefeat` | 失败结算，返回关卡选择 |

## 6. 轻Roguelike系统

游戏包含4个轻量Roguelike元素，在不增加过多复杂度的情况下提供变化性和重玩价值：

### 6.1 开局随机卡组

- 每关开始前，从玩家 **已解锁的永久卡池** 中随机抽取4张作为初始手牌
- 已解锁卡池 = 玩家已通关关卡奖励的卡牌集合（含初始默认卡）
- 每关有独立的卡池权重配置，保证主题一致性（如沙漠关倾向出现沙漠主题卡）

### 6.2 精英怪3选1

- 每波有1个精英怪（外观特殊、属性强化）
- 击杀后弹出3选1抽卡面板，从当前关卡卡池随机3张
- 如果手牌未满（<4张），选择的卡加入手牌
- 如果手牌已满（4张），玩家必须选择替换掉手牌中的一张
- 3选1面板出现时游戏暂停

### 6.3 关间Buff选择

- 通关后，从2个随机Buff中选择1个带入下一关
- Buff效果在该Run内持续（直到通关全部关卡或失败）
- 失败后所有Buff清零
- Buff示例：
  - "箭塔攻速+15%"
  - "所有士兵移动速度+20%"
  - "初始金币+50"
  - "水晶HP上限+100"
  - "技能卡伤害+25%"
- 关间Buff池详见 [04-levels §8](./04-levels.md#8-关间buff选择)

### 6.4 通关永久解锁

- 首次通关某关卡，解锁该关主题的1-2张新卡牌
- 新卡加入永久卡池，后续所有关卡的初始手牌抽取都会包含
- 解锁卡牌清单详见 [04-levels §7](./04-levels.md#7-通关解锁)

## 7. 与其他塔防的差异

| 对比 | 传统塔防（Kingdom Rush） | 卡牌Roguelike（怪物火车） | Tower Defender |
|------|------------------------|--------------------------|----------------|
| 部署方式 | 点工具栏图标→放地图 | 抽手牌→打出卡牌 | **拖手牌到地图格子** |
| 阵容来源 | 固定塔池 | 卡组构筑 | **开局随机抽+精英3选1补牌** |
| 经济 | 杀敌金币+被动产出 | 能量/费用 | **纯杀敌金币** |
| 关卡组织 | 独立关卡 | 长征Run | **5关顺序解锁，通关带Buff** |
| 重玩性 | 完美通关 | 卡组组合 | **随机手牌+Buff选择+永久解锁** |
| 升级 | 关内金币升级 | 关间强化 | **关内金币升级塔** |

---

## 8. 目标用户体验

- **单关时长**: 8-15分钟
- **通5关总时长**: 45-75分钟
- **上手难度**: 第1关简单，逐步引入新机制
- **核心用户**: 喜欢策略布阵 + 卡牌收集的玩家
- **情绪曲线**: 开局探索 → 中期运营 → BOSS决战 → 通关满足感
