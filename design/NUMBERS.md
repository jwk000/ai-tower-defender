## 1. 数值锚点与核心公式

### 1.1 基准锚点（箭塔 L1）

| 属性 | 值 | 语义 |
|------|-----|------|
| ATK | 11 | 基准攻击力 |
| AS | 1.0/s | 基准攻速 |
| DPS | 11 | 基准 DPS |
| 射程 | 200px | 基准射程（约 3.1 格）|
| 造价 | 50G | 基准建造成本 |
| DPS/G | 0.220 | 基准成本效率 |

### 1.2 核心公式

```
物理减伤 = 1 - 护甲 / (护甲 + 100)
魔法减伤 = 1 - 魔抗 / (魔抗 + 100)
实际攻速 = 基础攻速 × (1 + 攻速加成%)     上限 200%
实际移速 = 基础移速 × (1 + 移速加成%) × (1 - 减速%)   下限 20%
暴击倍率 = 1.5 + Buff暴伤加成
```

### 1.3 塔类平衡公式

```
有效 DPS = 单目标 DPS × AOE因子 × 射程因子 × 机制因子

AOE因子：
  单体: 1.0
  溅射(60%): 1.0 + 0.6 × 2 = 2.2
  链击(22%衰减, 3跳): 1.0 + 0.78 + 0.61 = 2.39
  贯穿(12.5%衰减, 3个): 1.0 + 0.875 + 0.766 = 2.64
  全图爆炸(120px): 1.0 + 1.0 × 3 = 4.0

射程因子 = (实际射程 / 200) ^ 0.5

目标 DPS/G 范围：
  单体DPS塔: 0.18~0.22
  AOE塔: 0.12~0.16
  纯效用塔: 0.08~0.12
  战略武器: 0.06~0.10
```
