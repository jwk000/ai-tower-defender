## 6. 第5关：深渊裂隙

### 6.1 基本信息

| 项目 | 内容 |
|------|------|
| 主题 | 深渊地下，黑暗压迫 |
| 天气 | Night（夜晚） |
| 波数 | 12波（第12波BOSS） |
| 水晶HP | 1500 |
| 难度 | 终极挑战 |

视觉要求：深渊裂隙必须启用棋盘月光（`map.lighting.moonlight.enabled: true`），以 shader 让棋盘向外发散较弱 bloom 光晕，但不在棋盘内部叠加提亮层或雾状遮罩；棋盘格子底色需要使用偏浅、低蓝度的深渊紫灰色，禁止使用过黑或过蓝的空地/路径色；飘荡微光粒子只出现在棋盘周围，不进入棋盘内部，必须保持低密度、破碎分布，不能形成连续白色环形边框，保证路径、空地和单位位置可读。

### 6.4 地图布局

```
路径走向：右侧3个出生点（上/中/下） → 3条路径 → 汇聚到中央通道 → 左侧水晶
路径长度：每条约12格，汇聚后约8格
空地分布：
  - 汇聚点前有少量空地（危险：离路径太近）
  - 地图上/下边缘有安全空地（远离所有路径，适合远程塔）
障碍物：漂浮石、紫色火焰、晶簇（中部分散）
```
