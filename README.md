# Tower Defender

**Tower Defender** 是一款卡牌驱动的关卡制塔防游戏。

在这里，防线不是开局就写死的建筑菜单，而是一手随战局变化的卡牌。你需要把塔、士兵、机关和技能拖进战场，在敌潮抵达水晶前搭起临时却可靠的防御阵线。每一次精英击杀都会带来新的 3 选 1 抉择，每一关胜利都会留下下一关可用的强化，让一局 5 关的推进充满变化。

设计文档入口见 [design/README.md](./design/README.md)。

## 一句话体验

拖卡布阵，守住水晶；击杀精英补强手牌，通关选择关间 Buff，在 5 个主题战场里一步步把随机开局打成完整流派。

## 核心玩法

### 拖卡上阵

你的手牌区固定展示 4 张卡。单位卡可以拖到路径周围的空地上，立刻变成塔、士兵或机关；技能卡可以拖到战场目标区域，直接释放爆发、控制或清场效果。

战斗不会因为你犹豫而停止。敌人沿路径不断推进，塔会自动攻击，士兵会顶上前线，机关会在关键格子制造机会。你要做的是在最短时间里判断：这一张牌该补输出、补控制，还是留给下一波压力？

### 水晶防守

水晶位于地图一侧，是整场战斗的最后防线。敌人一旦抵达水晶，就会持续造成伤害；水晶 HP 归零，本关失败。

胜利条件很直接：撑过全部波次，击败最后的 BOSS。难点在于，每一波都有不同速度、护甲、飞行、复活、自爆或召唤特性的敌人，单靠堆火力并不稳。

### 精英掉落

每波都会出现 1 个精英怪。击杀精英后，战斗暂停并弹出 3 张候选卡。手牌未满时直接补入；手牌已满时，你必须替换掉一张旧牌。

这让每一关都不是固定背板。你可能临时拿到炮塔补范围伤害，也可能拿到冰塔拖住 BOSS，还可能抽到奥术卡，把一条快崩的防线救回来。

### 关间成长

通关后，你可以从 2 个随机 Buff 中选择 1 个带入后续关卡。它们可能提升箭塔攻速、增加水晶 HP、强化魔法塔，也可能改变抽卡选择空间。

首次通关关卡还会解锁新的主题卡牌，加入永久卡池。越往后，卡池越丰富，组合也越容易打出自己的风格。

## 五个关卡主题

### 1. 绿野仙踪

雨中的草原是第一道战线，也是新手熟悉拖卡布阵的起点。哥布林、野猪和铁甲大象会逐步提高压力，最终由巨型史莱姆压轴登场。

这里的节奏相对温和，重点是学会把箭塔、冰塔、盾卫、弓手和基础机关组合成稳定防线。

### 2. 沙漠虫潮

烈日下的沙漠关主打数量压迫。沙漠黑虫成群涌来，钻地甲虫会短暂无敌，吸血蝗虫从低空掠过，自爆甲虫则会威胁你的塔群。

这一关鼓励你使用炮塔、火塔、毒塔和焦油坑处理密集敌人，在路径拐角处打出范围伤害。

### 3. 黑暗古堡

暗夜古堡开始引入更强的干扰能力。狼人冲锋，蝙蝠飞行，巫师施法，黑暗牧师还能复活倒下的敌人。最终 BOSS 路西法会把战场变成混乱的决战。

这一关考验目标优先级：什么时候先杀牧师，什么时候控住高血量怪，什么时候把输出集中到 BOSS 身上。

### 4. 末日废土

浓雾笼罩的废弃城市是机械军团的主场。机器狗、无人机、飞机、油罐车、坦克和巨型机器人轮番推进，地图还会出现分叉合流路径。

你需要同时管理多条防线，并在合流处构建核心火力。导弹塔、电塔和炮塔会成为这一关的关键牌。

### 5. 深渊裂隙

深渊裂隙是最终挑战。前三条路径同时出兵，前 4 关的敌人和 BOSS 会混合登场，最后由深渊领主守在终局。

这里不再考单一机制，而是考整局构筑：你带来的 Buff、解锁的卡牌、临场抽卡和布阵判断，都会决定水晶能不能撑到最后一波。

## 为什么每局都不一样

- **开局随机手牌** ：同一关也可能打出不同起手。
- **精英 3 选 1** ：每波都可能改变当前策略。
- **关间 Buff** ：胜利后的选择会影响后续关卡节奏。
- **永久解锁** ：通关越多，卡池越丰富。
- **主题敌人** ：每关的敌人组合和地图结构都在逼你调整打法。

## 技术实现

项目使用 **Vite + TypeScript** 构建，底层由 **bitecs ECS** 、配置驱动规则系统和 **PixiJS WebGL** 渲染组成。

技术方案的目标很简单：玩法内容尽量交给配置，系统代码负责稳定运行、规则调度和画面表现。更详细的架构说明见 [design/06-technical.md](./design/06-technical.md)。
