# 2026-06-18 开发日志

## 修复弩塔穿透弹误伤自身

- 问题：弩塔攻击敌人时自身会掉血。
- 原因：`ProjectileSystem` 的弩塔穿透弹分支只按 `Position + Health + UnitTag` 扫描沿途实体，并只检查层级可命中，没有校验发射源与目标阵营是否敌对；源塔和友方塔也可能被穿透线段命中。
- 修复：在公共 `canProjectileHitTarget` 中加入来源实体与目标实体的 `Faction` 敌对关系校验，确保有来源的弹道只能命中敌对单位。
- 回归：新增弩箭穿透测试，覆盖弩箭经过源塔、友方塔、敌人时只伤害敌人；同时让 ProjectileSystem 测试夹具补齐 `Faction`，更贴近真实运行实体。
- 验证：`npm test -- ProjectileSystem.test.ts` 通过；`npm run typecheck` 通过。
